# GCAF-Net

Official repository for:

**GCAF-Net: Geometry-Informed Cross-Attention Fusion Graph Neural Network for EEG-Based Diagnosis of Alzheimer’s Disease and Frontotemporal Dementia**

## 📄 Paper
[IEEE Xplore Link](https://ieeexplore.ieee.org/abstract/document/11460877)

## 🧠 Overview
We propose GCAF-Net, a geometry-informed graph neural network for EEG-based dementia classification. The model integrates spectral features, functional connectivity, and electrode spatial topology within an equivariant graph learning framework, where node coordinates are dynamically updated to capture brain network geometry. A bidirectional cross-attention mechanism is introduced to fuse spectral and spatial–functional representations, enhancing their interactions and complementarity. This design enables the model to learn more discriminative representations for accurate diagnosis of Alzheimer’s disease and frontotemporal dementia.
