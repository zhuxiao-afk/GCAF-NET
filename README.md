# GCAF-Net

Official repository for:

**GCAF-Net: Geometry-Informed Cross-Attention Fusion Graph Neural Network for EEG-Based Diagnosis of Alzheimer’s Disease and Frontotemporal Dementia**

## 📄 Paper
[IEEE Xplore Link](https://ieeexplore.ieee.org/abstract/document/11460877)

## 🧠 Overview
We propose GCAF-Net, a geometry-informed graph neural network that integrates spectral features, functional connectivity, and electrode spatial topology via cross-attention for EEG-based dementia classification.
